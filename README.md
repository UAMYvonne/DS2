# Customer Retention & Churn Analysis (Task 2)

##  Project Overview
This project involves a comprehensive analysis of customer churn within a telecommunications dataset. Using Power BI, I developed a dashboard to visualize key performance indicators (KPIs) and identify the underlying reasons why customers leave the service.

## Key Findings
* **Churn Overview:** 25.3% churn rate with 1,653 customers leaving.
* **Top Churn Driver:** Month-to-month contracts account for nearly 85% of all churned users.
* **Service Risk:** Fiber Optic customers are significantly more likely to churn than DSL users.
* **Revenue Loss:** Churned customers account for $2.8M in lost total charges.

##  Tools Used
* **Power BI:** Data visualization, DAX calculations, and dashboarding.
* **Data Cleaning:** Handled within Power BI (Power Query) to ensure accuracy.

## Action Plan
1. **Contract Strategy:** Convert monthly users to annual plans to stabilize revenue.
2. **Quality Audit:** Investigate the Fiber Optic service experience to reduce segment-specific churn.
3. **Payment Optimization:** Encourage a move from Electronic Checks to Automated Credit Card payments.
